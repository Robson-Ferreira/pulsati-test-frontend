# React Application

## Descrição

Esta é uma aplicação React que inclui uma tela de login e um dashboard. Siga as instruções abaixo para configurar e rodar o projeto em seu ambiente local.

## Pré-requisitos

- Node.js e npm (ou yarn)

## Instalação

1. **Instale as dependências do projeto:**

    ```bash
    npm install
    # ou, se você estiver usando yarn
    yarn install
    ```

## Executando a Aplicação

1. **Inicie o servidor de desenvolvimento:**

    ```bash
    npm start
    # ou, se você estiver usando yarn
    yarn start
    ```
